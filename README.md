# Koma

A React Native / Expo app for learning Turkish classical music — makams (melodic modes), usul (rhythmic patterns), and rhythm training.

> **Koma** is the smallest interval in Turkish music theory, representing the subtle microtonal differences that make this tradition unique.

## Features

- **Makams** — Browse 16 Turkish makams with detailed descriptions, scales, and characteristics
- **Usul (Rhythms)** — Explore 11 Turkish rhythmic patterns with time signatures and beat breakdowns
- **Metronome Trainer** — Tap-based rhythm accuracy tool with real-time BPM calculation, color-coded feedback, and scoring
- **i18n** — Full Turkish and English language support
- **Authentication** — User accounts via Supabase (email/password)

## Tech Stack

- React Native 0.76 + Expo SDK 52
- React Navigation 6 (Material Top Tabs)
- Supabase Auth
- AsyncStorage for user preferences
- Animated API for UI animations

## Getting Started

```bash
npm install
npm start
```

Then press `i` for iOS, `a` for Android, or `w` for web.
